# Sentinel Theme

> A comprehensive theme collection bringing Sentry's distinctive purple and pink brand colors to Visual Studio Code.

## Features

- **Three beautiful variants**: Light, Dark, and Midnight
- **WCAG AA accessibility compliant** colors for optimal readability
- **Carefully crafted** using Sentry's official brand palette
- **Comprehensive coverage** for syntax highlighting, UI elements, and terminal

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Sentinel Theme"
4. Click Install
5. Go to File > Preferences > Color Theme and select your preferred Sentinel variant

## Variants

### Sentinel Light
A clean, bright theme perfect for daytime coding with excellent contrast.

### Sentinel Dark
A balanced dark theme with Sentry's signature purple and pink accents.

### Sentinel Midnight
An ultra-dark theme for late-night coding sessions with minimal eye strain.

## About

This theme is designed with developer ergonomics in mind, ensuring optimal readability and visual hierarchy while maintaining Sentry's distinctive brand identity.
